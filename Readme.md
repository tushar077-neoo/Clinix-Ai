# 🧠 Clinix AI - Intelligent Health Screening System

Clinix AI is an AI-powered health assistant that predicts possible diseases using user symptoms and blood parameter inputs, and suggests nearby clinics using Google Maps integration.

## 🚀 Features

* 🧪 Blood parameter-based health analysis
* 🤖 Disease prediction using Machine Learning
* 📊 Model accuracy evaluation
* 📍 Real-time clinic suggestions via Google Maps API
* 🌐 Interactive UI using Streamlit

## 🧠 How It Works

1. User enters symptoms and blood report values
2. Machine learning model predicts possible disease
3. Displays prediction confidence
4. Suggests nearby clinics using location data

## 🛠️ Tech Stack

* Python
* Scikit-learn
* Pandas
* Streamlit
* Google Maps API
